### 项目运行

    npm install 或 cnpm install
    npm run dev //开发dev
    npm run build //开发打包
    npm run pro //生产打包

### 项目结构：
+ 目录src：存放js 下面分文件夹
+ 目录static：存放静态资源
    * css: 存放样式
    * fonts: 字体库
    * img: 存放图片
    * html: 存放html
    * template: 存放模板
+ 目录assets：开发打包目录（勿提交）
+ 目录dist：生产打包目录
+ 目录libs：公共模块
+ 目录test：测试目录

### 项目规范
* 改动公共文件请通知组内人员

### 文件命名
* 所有的HTML标签小写，文件名称小写
* js方法采用驼峰命名
* 类首子大写
